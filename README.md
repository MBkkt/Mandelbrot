# Mandelbrot

Simple mandelbrot viewer

![ ](https://raw.githubusercontent.com/MBkkt/Mandelbrot/master/bin/mandelbrot.jpg)

| Button | Explain    |
| :----- | ---------: |
| W      | move up    |
| A      | move left  |
| S      | move down  |
| D      | move right |
| E      | zoom in    |
| Q      | zoom out   |
| Esc    | exit       |
